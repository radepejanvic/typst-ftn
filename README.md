U `zavrsni-rad.pdf` se nalazi kompajlirana pdf verzija rada. 
U direktorijumu `./poglavlja` se nalazi izvorni tekst svakog poglavlja. 

Koraci za rad na repozitorijumu: 
- [instalacija](https://typst.app/open-source/) `typst` programa
- otvoriti repozitorijum u VS code-u, jer ima PDF pregledac koji automatski azurira PDF prilikom svakog cuvanja
- pokrenuti komandu `typst watch ./zavrsni-rad.typ` kako bi se promene azurirale prilikom svakog cuvanja
